# MailDaemon.Core

MailDaemon.Core is a modern .NET 9 class library for batch mail processing and automation.
It fully replaces the legacy MailDaemon.Lib project, which was previously based on .NET Standard.

## 🚀 Overview

MailDaemon.Core serves as the new, maintained, and extensible core library of the MailDaemon ecosystem.
It is designed to provide a stable, high-performance foundation for bulk email operations while leveraging the latest features of .NET 9.

## 🧩 Key Improvements

- Migration from .NET Standard to .NET 9.
- Improved architecture and performance.
- Support for modern .NET features (async/await, dependency injection, logging, configuration).
- Easier integration with other MailDaemon services and applications.
- Better separation of concerns and improved testability.
